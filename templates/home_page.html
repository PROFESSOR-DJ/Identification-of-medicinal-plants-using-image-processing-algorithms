HOW IT WORKS ?

1. Upload an image of the plant/leaf to identify and gather medicinal information. ( Must be of jpg, jpeg or png format )
2. Image processing and labeling of the corresponding plant species takes place.
3. Feature extraction and recognition using Convolutional Neural Networks ( CNNs ) trained models.
4. Display of plant name with basic medicinal information ( i.e. dosage, action,ayurvedic names, family, habitat, etc. ) having a high accuracy.
